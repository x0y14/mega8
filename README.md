# mega8
ロックマン８の素材を使用して、ロックマンを自作を試みるもの。


## usage
#### unix
1. `go mod tidy`
2. `cd scripts`
3. `make build`
4. `./bin/mega8`
---
#### windows
[make](http://gnuwin32.sourceforge.net/packages/make.htm) をインストールし、pathを通す必要があります。
1. `go mod tidy`
2. `cd scripts`
3. `make build`
4. `./bin/mega8.exe`
